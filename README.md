# 曙光 Never End

虽然曙光在未来的某天游戏化时，或许并没有真正意义上的游玩顺序，但对于游玩体验来说，还是献上了个人建议来的UwU：  


**终末之章**（霁雪侧/化雨侧可不分先后）→**往昔篇/除去淡墨的各种支线** →**前世篇/淡墨支线**（包括~~双神发糖~~特殊支线） →**遗迹篇**
（也可以在终末之章全结局达成之后游玩淡墨支线，然后游玩遗迹篇XD  
~~（又名：恰 粮 之 路（？）~~
