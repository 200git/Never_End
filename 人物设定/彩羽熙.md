# 彩羽熙/Criaxi

性别：无性

年龄：未知

（tips：轮回的年龄是可以叠加的( ᐛ )）

种族：羽狐族

〖简介〗

  彩羽熙原本是直接属于淡墨管理的造物之一，在那期间，他创始了羽狐族，并被族人尊称为神。

 淡墨失踪后，彩羽熙开始质疑在这个世界所做的一切是否有意义，便退隐于镜中，托身于梦里，直到现在仍不知去向……

〖彩羽熙的喜好〗

  “啊，下官平时喜欢占卜和绘画，对于植物和宝石晶体一类也是有些留意的，晶藤那时特别兴奋地和我说我会喜欢的，但……到现在我也没在其中找到兴趣。”

〖彩羽熙讨厌的事物〗

  “淡墨大人的好友岚影……虽然同样是神，但的确是个表里不一的人，很危险。改天要和她说下需要多加提防。”

  “下官并不喜欢除了鸟以外的肉类……啊，很意外吗？但下官真的不能下口……”

  “……为什么会有柠檬这种东西的存在啊！下官真的不能理解！！”

〖彩羽熙的日常工作〗

  彩羽熙的工作与其他神使不同，本应由白暮负责的“和平”一事由他接手，简单来说，就是调节，避免整个世界的战争。

  这份工作似乎并不算累，但好像也不轻松？

〖彩羽熙的能力〗

  彩羽熙的“三色羽”能够带来不同的好运，也能够了解关于这方面的过去，现在与未来

 粉红色的羽毛代表人的一生

 黄色的羽毛含义为人的性格

 天蓝色的羽毛预示人的思想

 通过“三色羽”，彩羽熙就能够了解某人在这三方面的变化

〖彩羽熙的铃铛〗

  彩羽熙所戴的铃铛是汐做的，从他被创造出来时就有了。

  “这个，就送给彩羽熙吧！”

 “嗯……确实很适合他呢……他应该会喜欢的……”

  淡墨将彩羽熙的灵魂寄存在了铃铛上，当然，这并不代表铃铛坏掉后彩羽熙就消逝了。

  和那位创造者不同，彩羽熙是真正意义上的永生的，除非自己想要死去。

〖彩羽熙和镜中梦〗

 镜中梦其实是彩羽熙创造的空间，所用方法与冰梦创造空间时相似，都是将魔力注入某件物品后扩散得到的。

 镜中梦类似一个“镜子迷宫”，在其中毫无生命迹象，其中的所有事物几乎全是由镜子构成的。由于彩羽熙本身并不需要生命活动就能存活，主线中就一直在镜中梦休眠……

〖关于三彩祭〗

  “三彩祭”传说是羽狐族用来向彩羽熙寻求平安和好运的祭祀，由族中最有威望和能力最强的族人交替主持，但可能也会有两者为同一人的情况出现？

  现在的三彩祭，正是由魔力强大的利格特尔主持，或许在梦中，彩羽熙仍能感受到羽狐族的祈祷吧……

〖RE线结局〗

  真结局的彩羽熙已经醒来，并且走出了困居自己的空间，重新接手自己所做的工作，虽说这个世界已经和他最后见到的景象大相庭径，淡墨大概也会会带他适应的……

〖BE线结局〗

  坏结局中，身为神职的利格特尔开始了弑神工作，她来到了镜中梦，唤醒了沉睡的彩羽熙，以当时她所看到的景象，诱导着使彩羽熙开始消逝……

  不知道他在最后的那一刻，有没有对自己所做的一切感到失望呢？

〖各人对彩羽熙的评价〗

（tips：评价并不是来自同一时间段）

 “彩羽熙很关心他人的……对我也一样……不过他似乎很反感岚影……究竟是为何，我也没深究过……” 

〔岚影〕

 “为什么……彩羽熙不愿意和我交流呢quq……是我做错了事情吗？如果是这样的话他为什么不和我说呢〒▽〒……” 

〔冰梦〕

  “谁？”

（tips：真结局之前冰梦并没见过彩羽熙）

〔晶藤〕

 “彩羽熙前辈？他是个很温柔的人，当时他教过我一些占卜的知识来着，不知道他现在去了哪里……呃，有没有可能，他已经忘记我了……不管怎么样，希望我们还能再次相见吧。” 

〔白暮〕

〔安格〕
